Leviora Edu: Technical Roadmap & Architecture

A detailed overview of the platform's core features, technology stack, and development plan.

Project Overview

  • Objective: Build India’s first skills-first ecosystem connecting learning, mentorship, projects, jobs, and global careers.
  • Target Users: Students, Professionals, and Companies.
  • Main Features: Live Classes + Community + AI-driven Career Guidance.
  • Goal: Bridge the gap between education and employability.

Development Roadmap

Phase 1 (3–4 months): MVP

  • User Authentication & Profiles
  • Skill Dashboard (Basic)
  • Live Study Rooms (Video, Chat, Whiteboard)
  • Community Chat
  • Basic Skill Courses (Live + Recorded)

Phase 2 (6–8 months): Growth

  • AI Career Counselor
  • Job Board + Portfolio Builder
  • Mentorship System (1:1 & group)
  • AI Learning Paths
  • Advanced AI Recommendations

Phase 3 (1 Year+): Scale & Expansion

  • Global Networking & Hiring Platform
  • Corporate Training Modules
  • Advanced analytics dashboards

Technical Architecture

Frontend

Flutter (mobile), React.js/Next.js (web)

Backend

Node.js + Express/NestJS, PostgreSQL/MySQL, Firebase/Supabase, Redis

Live Classes

WebRTC, Jitsi/Twilio/Agora, Socket.IO/Firebase chat

AI Features

Python + PyTorch/TensorFlow/OpenAI APIs for career guidance

Cloud & Hosting

AWS/GCP, S3/Cloud Storage, CDN

Security

JWT authentication, Role-based access, Encryption

API & Data Requirements

  • APIs: User, Courses, Live Classes, Mentorship, Portfolio, Job Board
  • Database Tables: Users, Roles, Courses, Projects, Mentorship Sessions, etc.

Tech Risks & Considerations

  • Scalability: For live classes and AI features.
  • Latency: Target ≤200ms for real-time interactions.
  • Data Privacy: Adherence to GDPR and Indian laws.
  • AI Accuracy: Implementing feedback loops for career recommendations.

Tools & Collaboration

  • Project Management: Jira / ClickUp
  • Version Control: GitHub / GitLab
  • CI/CD: GitHub Actions / Jenkins
  • Design & Prototypes: Figma / Canva
  • Analytics: Google Analytics, Mixpanel, Firebase Analytics

Deliverables for Meeting

  • Technical Architecture Diagram
  • MVP Feature List + Tech Stack
  • Roadmap (MVP → Full Product)
  • User Journey Charts
  • API & Database Overview
  • Risks & Mitigation Plan
